Method 1 - from ozzmaker.com¶

Color Escape Codes¶
In [ ]:
print("\033[1;32;48m Bright Green  \n")

Entire Color Code = \033[1;32;48m

-- Escape code = \033[

-- Text Style = 1;

-- Text Color = 32;

-- Background Color = 48m (48 is not on the Background Color chart below)

Examples¶
In [ ]:
print("\033[1;31;48m Red  \n")              # RED text color
print("\033[1;30;48m Hello World!  \n")     # BLACK text color

Method 2 - from Haoyi's Programming Blog page titled: Building your own Command Line with ANSI escape codes.¶

In [ ]:
print("\u001b[31mHello World")

Entire color escape code = \u001b[31m

In [1]:
print("\u001b[31mHello World")
print("Welcom, Dave.")

Hello World
Welcom, Dave.
Change and Reset¶
In [2]:
print("\u001b[31mHelloWorld\u001b[0m")
print("Welcom, Dave.")

HelloWorld

import sys
for i in range(0, 16):
    for j in range(0, 16):
        code = str(i * 16 + j)
        sys.stdout.write(u"\u001b[38;5;" + code + "m " + code.ljust(4))
    print(u"\u001b[0m")
