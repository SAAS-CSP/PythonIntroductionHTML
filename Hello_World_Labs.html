1-3.2a Create the pWelcome() Function¶

A function is a group of statements. Even though the Welcome message is only one print() statement, it is enough for a function to work properly.

Below is an example for creating a function. Start with the 'def' keyword followed by the name of the function. I like to give my function names an easy name to remember. For example, since these functions are for printing I like to start them with the letter ‘p’. Lastly, end the function declaration statement with parentheses. That is it for now, later we will look at adding arguments to functions.

In [8]:
def pWelcome():    #2a. Welcome Message Function
    print("Hello and welcome to my app!")
Be Aware: Notice line #2 has an indention. There are two spaces before the word print on line number two. Make sure your statements are lined up correctly. Python will throw an indentationError Exception if they are not indented correctly. The easiest way to get correct indents is to place the cursor at the end of line 1. Hit the return key and it will automatically indent correclty.
